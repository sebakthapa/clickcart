
# ClickCart

This forked repo is a technical assessment for Creating basic E-commerce website.


## Run Locally

Clone the project

```bash
  git clone <https://link-to-project>
```

Go to the project directory

```bash
  cd <project-name>
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
